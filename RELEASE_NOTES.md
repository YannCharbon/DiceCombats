# Release notes

### New

- **Support of aidedd.org DnD 5.5 stat block extraction**
  You can now import a DnD 2024 creature stat block from aidedd.org, DnD 2014 was already supported.

### Improvements

- Nothing

### Bug fixes

- **Aidedd.org stat block extraction issue**
  Aidedd.org website changed their way of managing CSS styles of their stat blocks which caused rendering issues in DiceCombats.
